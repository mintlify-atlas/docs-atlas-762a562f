GREL provides a full suite of mathematical functions, from basic arithmetic to trigonometry and combinatorics. All math functions accept `number` arguments and return `number` unless noted. GREL numbers are double-precision floating-point by default; integer-returning functions (like `ceil`, `floor`, `round`) return a `long` integer when the result fits.

## Absolute value and rounding

| Function | Signature | Returns | Description |
|---|---|---|---|
| `abs` | `number n` | number | Returns the absolute value of `n` |
| `ceil` | `number n` | number | Returns the smallest integer greater than or equal to `n` (ceiling) |
| `floor` | `number n` | number | Returns the largest integer less than or equal to `n` (floor) |
| `round` | `number n` | number | Returns `n` rounded to the nearest integer (half-up) |
| `even` | `number n` | number | Rounds `n` up to the nearest even integer |
| `odd` | `number n` | number | Rounds `n` up to the nearest odd integer |

```text
abs(-5)       // 5
abs(3.7)      // 3.7
ceil(3.2)     // 4
floor(3.9)    // 3
round(3.5)    // 4
round(3.4)    // 3
even(3)       // 4
even(4)       // 4
odd(4)        // 5
odd(5)        // 5
```

## Arithmetic and division

| Function | Signature | Returns | Description |
|---|---|---|---|
| `mod` | `number n1, number n2` | number | Returns `n1` modulus `n2` |
| `quotient` | `number numerator, number denominator` | number | Returns the integer quotient of the division (truncates toward zero) |
| `pow` | `number n1, number n2` | number | Returns `n1` raised to the power of `n2` |

```text
mod(10, 3)        // 1
mod(7.5, 2)       // 1.5
quotient(10, 3)   // 3
pow(2, 10)        // 1024.0
pow(9, 0.5)       // 3.0
```

## Min and max

| Function | Signature | Returns | Description |
|---|---|---|---|
| `min` | `number n1, number n2` | number | Returns the smaller of two numbers |
| `max` | `number n1, number n2` | number | Returns the larger of two numbers |

```text
min(3, 7)     // 3
max(3, 7)     // 7
```

## Logarithms and exponentials

| Function | Signature | Returns | Description |
|---|---|---|---|
| `ln` | `number n` | number | Returns the natural logarithm (base *e*) of `n` |
| `log` | `number n` | number | Returns the base-10 logarithm of `n` |
| `exp` | `number n` | number | Returns *e* raised to the power of `n` |

```text
ln(1)          // 0.0
ln(2.718281828)  // ≈ 1.0
log(100)       // 2.0
log(1000)      // 3.0
exp(1)         // 2.718281828...
exp(0)         // 1.0
```

## Factorials and combinatorics

| Function | Signature | Returns | Description |
|---|---|---|---|
| `fact` | `number n` | number | Returns `n!` (n factorial), equivalent to `factN(n, 1)` |
| `factN` | `number n1, number n2` | number | Returns the multifactorial: product of every `n2`-th integer from `n1` down to 1. E.g., `factN(9, 3)` = 9 × 6 × 3 = 162 |
| `combin` | `number n1, number n2` | number | Returns the number of combinations of `n2` items chosen from `n1` (binomial coefficient C(n1, n2)) |
| `multinomial` | `number n1, number n2, ...` | number | Returns the multinomial coefficient for the given set of numbers |
| `gcd` | `number n1, number n2` | number | Returns the greatest common divisor of two numbers |
| `lcm` | `number n1, number n2` | number | Returns the least common multiple of two numbers |

```text
fact(5)            // 120  (5! = 5×4×3×2×1)
fact(0)            // 1
factN(10, 2)       // 3840 (10×8×6×4×2)
combin(5, 2)       // 10   (C(5,2) = 10)
combin(10, 3)      // 120
multinomial(1,2,3) // 60
gcd(12, 8)         // 4.0
lcm(4, 6)          // 12.0
```

## Trigonometry

All trigonometric functions operate in **radians**. Use `degrees()` and `radians()` to convert.

| Function | Signature | Returns | Description |
|---|---|---|---|
| `sin` | `number n` | number | Trigonometric sine of angle `n` (radians) |
| `cos` | `number n` | number | Trigonometric cosine of angle `n` (radians) |
| `tan` | `number n` | number | Trigonometric tangent of angle `n` (radians) |
| `sinh` | `number n` | number | Hyperbolic sine of `n` |
| `cosh` | `number n` | number | Hyperbolic cosine of `n` |
| `tanh` | `number n` | number | Hyperbolic tangent of `n` |
| `asin` | `number n` | number | Arc sine of `n`, in the range [-π/2, π/2] |
| `acos` | `number n` | number | Arc cosine of `n`, in the range [0, π] |
| `atan` | `number n` | number | Arc tangent of `n`, in the range [-π/2, π/2] |
| `atan2` | `number n1, number n2` | number | Arc tangent of the point (n1, n2), converting rectangular to polar coordinates; returns angle theta |
| `degrees` | `number n` | number | Converts `n` from radians to degrees |
| `radians` | `number n` | number | Converts `n` from degrees to radians |

```text
sin(0)             // 0.0
cos(0)             // 1.0
tan(radians(45))   // ≈ 1.0
asin(1)            // ≈ 1.5707963... (π/2)
acos(0)            // ≈ 1.5707963... (π/2)
atan(1)            // ≈ 0.7853981... (π/4)
atan2(1, 1)        // ≈ 0.7853981... (π/4)
degrees(3.14159)   // ≈ 180.0
radians(180)       // ≈ 3.14159...
```

## Random numbers

| Function | Signature | Returns | Description |
|---|---|---|---|
| `randomNumber` | `number lowerBound, number upperBound` | number | Returns a random integer in the range [`lowerBound`, `upperBound`] (both inclusive). Each row receives a different random value. |

```text
randomNumber(1, 100)   // a random integer between 1 and 100 inclusive
```

<Warning>
`randomNumber()` produces a different value for each row every time you run a transform. If you need stable random numbers, store the result in a new column first.
</Warning>

## Sum over arrays

| Function | Signature | Returns | Description |
|---|---|---|---|
| `sum` | `array a` | number | Returns the sum of all numeric values in the array. Non-numeric items are ignored; returns 0 for an empty or all-non-numeric array. |

```text
[1,2,3,4].sum()          // 10
[1,"x",3,null].sum()     // 4
```

## Practical examples

**Clamp a value between 0 and 100:**
```text
max(0, min(100, toNumber(value)))
```

**Round to two decimal places:**
```text
round(toNumber(value) * 100) / 100
```

**Compute the hypotenuse of a right triangle:**
```text
pow(pow(toNumber(cells["A"].value), 2) + pow(toNumber(cells["B"].value), 2), 0.5)
```

**Normalize a score to a 0–1 range using min/max from constants:**
```text
with(toNumber(value), v, (v - 0) / (100 - 0))
```

**Check whether a value is divisible by 3:**
```text
mod(toNumber(value), 3) == 0
```
